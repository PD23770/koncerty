!!!! Uruchomienie projektu !!!!

Należy uruchomić lokalnie w środowisku Jupyter Notebook lub JupyterLab. Plik "dane.csv" musi znajdować się w tym samym folderze co notebook.

Wykresy zostały wykonane w Plotly. Na stronie GitHub mogą wyświetlać się jako ograniczony podgląd.


1) Wizualizacja rynku koncertów muzycznych w Polsce

Projekt przedstawia analizę syntetycznych danych o 1200 koncertach muzycznych w Polsce w latach 2024–2025.

2) Technologie

Python
pandas
Plotly
Jupyter Notebook

3) Pliki

- "wizualizacja-koncerty.ipynb" to notebook z analizą i interaktywnymi wykresami.
- "dane.csv" to dane wykorzystane w analizie.

4) Zakres analizy

Notebook zawiera:

1. Wczytanie i wstępną eksplorację danych.
2. Wykres słupkowy łącznego przychodu według miasta.
3. Dwa wykresy liniowe pokazujące liczbę koncertów w czasie.
4. Histogram cen biletów i boxplot przychodów według typu obiektu.
5. Scatter plot ceny biletu i wypełnienia sali.
6. Mapę koncertów w polskich miastach.
7. Kompozycję czterech wykresów w układzie 2 × 2.
8. Wnioski z analizy rynku koncertowego.
